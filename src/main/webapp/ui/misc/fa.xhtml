FontAwesome Font Awesome is a highly customizable scalable vector iconset with 605 icons. In addition to the default ThemeRoller icons, Font Awesome icons are provided out of the box within PrimeFaces. In order to enable Font Awesome support, enable primefaces.FONT_AWESOME setting as true via a context param.

For the complete list of icons, visit font awesome demo.
